# Computer Science Journey: From Logic to Application

這是我在 **國立金門大學 (NQU)** 大一期間的程式設計學習紀錄。本倉庫不僅是我的作業存檔，更記錄了我從基礎過程導向語言 (C) 銜接至現代物件導向開發 (C# / WPF) 的技術成長曲線。

---

## 大一上：C 語言邏輯與演算法實作
專注於邏輯思維的磨練，我認為「嚴謹的邏輯掌控」是資工系的靈魂。

### 技術亮點 (Highlights)
* **02_Math_Algorithms**: 實作質數、完美數、阿姆斯壯數。練習迴圈優化與邊界條件處理。
* **03_Recursion**: 深度實作「河內塔」與「費氏數列」。理解 **分治法 (Divide and Conquer)** 與遞迴調用機制。
* **07_Sorting_Algorithms**: 實作「氣泡排序 (Bubble Sort)」，掌握基本的資料排序邏輯與 $O(n^2)$ 演算法複雜度概念。
* **04/05/06**: 涵蓋字串處理、複雜巢狀迴圈圖形控制以及陣列操作。

## 大一下：C# & WPF 現代化桌面開發 (Current)
目前正透過課程指定教材 **《Pro WPF 4.5 in C#》** 踏入 Windows 應用程式開發領域。這是我從「結構化程式設計」跨越到「物件導向程式設計 (OOP)」的關鍵階段。

### 核心學習進度
* **物件導向思維**: 學習類別 (Class)、物件 (Object) 與封裝概念。
* **Visual Studio 實作**: 掌握專業 IDE 的專案管理、方案配置與除錯 (Debug) 技巧。
* **XAML 介面設計**: 探索如何使用標記語言設計向量介面、佈局 (Layout) 與控制項應用。

> ** 狀態更新**: 正在深入學習 WPF 的相依屬性與資料綁定基礎，專案原始碼將隨著課程進度同步於 `08-XX` 之後的目錄更新。
