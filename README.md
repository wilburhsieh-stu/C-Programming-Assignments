# Computer Science Journey: From Logic to Application

這是我在 **國立金門大學 (NQU)** 資訊工程學系大一期間的程式設計學習紀錄。本倉庫不僅是我的作業存檔，更記錄了我從基礎過程導向語言 (C) 銜接至現代物件導向開發 (C# / WPF) 的技術成長曲線。

---

## 大一上：C 語言邏輯與演算法實作
專注於邏輯思維的磨練，我認為「嚴謹的邏輯掌控」是資工系的靈魂。

### 技術亮點 (Highlights)
* **02_Math_Algorithms**: 實作質數、完美數、阿姆斯壯數，練習迴圈優化與邊界條件處理。
* **03_Recursion**: 深度實作「河內塔」與「費氏數列」，理解 **分治法 (Divide and Conquer)** 與遞迴調用機制。
* **07_Sorting_Algorithms**: 實作「氣泡排序 (Bubble Sort)」，掌握基本的資料排序邏輯與 $O(n^2)$ 演算法複雜度概念。
* **04/05/06**: 涵蓋字串處理、複雜巢狀迴圈圖形控制以及陣列操作。
