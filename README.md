# Computer Science Journey: From Logic to Application

這是我在國立金門大學大一期間的程式設計紀錄。這個倉庫不僅是我的作業存檔，更記錄了我從基礎過程導向語言（C），逐漸接觸物件導向開發與現代化介面設計（C# / WPF）的探索過程。

## 大一上：C 語言邏輯 (C Programming)
大一上學期，我專注於鍛鍊邏輯思維與基礎演算法。我認為資工系的靈魂在於對邏輯的嚴謹掌控。

### 學習分類與亮點
- **02_Math_Algorithms**: 實作質數、完美數、阿姆斯壯數等，在此練習了迴圈優化與邊界條件處理。
- **03_Recursion**: 實作「河內塔」與「費氏數列」，這讓我對遞迴（Recursion）的運作機制與分治法（Divide and Conquer）有深刻理解。
- **07_Sorting_Algorithms**: 實作「氣泡排序 (Bubble Sort)」，掌握基本的資料排序邏輯。
- **04/05/06**: 涵蓋了字串處理、複雜迴圈圖形控制以及陣列操作。

## 大一下：C# & WPF 開發 (In Progress)
目前在大一下學期，我正開始接觸現代化桌面應用程式開發。並由**《Pro WPF 4.5 in C#》**作為教材，雖然目前還處於起步階段，但我正致力於掌握：

- **物件導向思維**: 從 C 的結構化思維轉向 C# 的物件導向 (OOP) 邏輯。
- **Visual Studio 實作**: 學習使用專業 IDE 管理方案（Solution）與專案（Project）。
- **XAML 基礎**: 探索如何使用標記語言設計向量介面與佈局（Layout）。

> **目前的進度**: 正在學習 WPF 的環境配置與基本控制項使用，這部分的專案碼將隨著學習進度陸續更新。
